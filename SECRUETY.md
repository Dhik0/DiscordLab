How it works:
This tool is a visual customizer for BetterDiscord. It operates by generating CSS files and syncing them with your local Discord folder.

Your Data:

Open Source: The entire logic is written in Python and is fully transparent.

Zero Data Collection: No personal data, Discord tokens, or passwords are ever accessed or uploaded.

Executable Safety: The .exe version is bundled via PyInstaller. Some antiviruses may flag it as a "False Positive" due to the bundling process. You are encouraged to run the script directly from source if you have concerns.

Reporting a Vulnerability:
If you find any security issues, please open an Issue or contact me directly in my Discord Lab.
